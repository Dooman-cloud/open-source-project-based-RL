
# 통계-AI 모델 기반 일 별 금융 자산 리스크 관리 플랫폼

## 1. 프로젝트 배경

#### **기존 서비스의 한계**

금융 리스크 분석은 RSI, MA 등 통계적 기술 지표에 의존하는 경우가 많으며, 이러한 수치는 일반적인 개인 투자자가 쉽고 직관적으로 이해할 수 없다.

AI 기반(RL, DL 등) 리스크 예측 일반화 모델은 금융 시계열 데이터의 비정상성과 종목 간 이질성과 같은 challenging point로 인해 모델의 방대한 학습 시간, 막대한 자원, 데이터 센터와 GPU 등의 고급 하드웨어를 필요로 한다. 이 일반화 모델을 실시간으로 운용하기 위한 재학습 비용은 천문학적일 뿐 아니라 운용이 된다고 가정해도 모델이 BlackBox이기 때문에 그 결과의 해석력도 매우 낮다.

#### **서비스 개요**

본 서비스는 통계 모델을 기반으로 주가 리스크를 분석하고, 그 결과를 챗봇과 직관적인 UI를 활용하여 **사용자가 쉽게 이해할 수 있는 형태로 제공하는 실용적인 금융 리스크 관리 서비스를 구축**하는 것을 목표로 한다.

이는 통계 모델의 큰 장점인 높은 해석력을 기반으로 그 분석 결과를 LLM이 쉽게 해석함으로 개인 투자자의 합리적인 의사 결정을 지원하며, 모델 운용 비용을 최소화하여 무료 서비스로 배포할 수 있다. 

#### 기대 효과

- **쉽고 직관적인 정보 제공**
→ 위험 종목 랭킹 + 챗봇 + 분석 결과 자연어 요약 설명
- **일반 개인 투자자의 의사결정 지원**
→ 간단하지만 투자 리스크 관리에서 필수적인 지표들을 무료 서비스로 제공
- **분석 시간 단축**
→ 여러 지표를 한 화면에서 요약 및 통합 디스플레이

## 2. 주요 기능

#### 1) VaR 기반 리스크 대시보드

- **Stock Price with GARCH VaR Chart**
주가 차트 위에 VaR 라인을 오버레이하여 시각화

- **Volatility Chart**
 추정된 변동성(σ)을 차트로 제공

- **Statistical Indicator Chart**
RSI, MA, BB  등 기본 보조 지표 제공 (보조 역할)

#### 2) 리스크 랭킹 시스템

높은 위험성 상위 종목 3개 디스플레이함으로 사용자가 현재 고위험군 종목을 빠르게 파악 가능

ex)

- 분산 변동성이 큰 종목
- VaR이 큰 종목

#### 3) 챗봇 리스크 해석 서비스

VaR/주가 변동성 등 리스크 분석 결과에 대한 지표를 쉽게 이해할 수 있는 자연어로 설명. 

**EX 1)**

**사용자**: “내일 KOSPI 200 손실액 분석”

**챗봇**: “KOSPI 200의 ES(Expected Shortfall)영역을 보면, 평균 손실액이 전일 대비 1.8% 증가하여 자본 운영에 대한 주의가 필요합니다. 해당 종목 포트폴리오 비중을 5% 이상 낮추는 것을 권장합니다.”

**EX 2)**

**사용자:** “과거 데이터 대비 S&P 500의 위험 수준”

**챗봇:** “현재 S&P 500의 수익률 변동성은 과거 대비 상위 10%입니다”

#### 4) 프로필 설정 기능

사용자가 설정한 투자 성향, 분석 기간, 종목에 따라 리스크 분석 모델 조정.

- VaR 신뢰수준 조정
안정형 투자자 / 공격형 투자자 (α = 0.01 / 0.05)
- 종목 선택
삼성전자, KOSPI 200, TOPIX,  …
- 분석 기간
1주일~10년

#### 5) 리스크 분석 결과 요약

분석 결과를 자연어 형태로 요약

- 자본 기반 VaR 요약:

→ VaR: -3.2% (95%) : 유의수준 5%로 투자금 1,000만원 기준 약 32만원 손실 가능

- 리스크 변동성 요약:

→ 전일 대비 +18% 증가

→ 최근 5일 평균 대비 +25%

## 3. 사용자 경험 시뮬레이션

1. **플랫폼 접속 및 종목 선택**
    - 관심 종목 선택 (KOSPI200, S&P500, …)
    - 투자 성향에 따라 신뢰수준 선택 
    안정형 투자자 / 공격형 투자자 (α = 0.01 / 0.05)
    - 분석 기간 설정 (1주일~15년)
2. **대시보드 확인**
    - 주가 VaR 기반 리스크 차트
    - 변동성 차트
    - 통계 / 경제 지표
3. **리스크 예측 분석 결과 리포트**
    - 미래 VaR 예측값
    - 미래 주가 변동성
4. **리스크 랭킹 차트 알림**
    - 주가 변동성 높은 종목 TOP 리스트
    - VaR이 높은 종목 TOP리스트
5. **챗봇 대화**
    - 통계 / 경제 지표를 자연어로 쉽게 풀어 설명
6. **의사결정**
    - 사용자 포트폴리오 조정

## 4. 시스템 아키텍처

### 1) 데이터 소스 → Backend

- `yfinance` API를 통해 일별 주가 데이터 수집

### 2) Backend

1. **Python 모델링** 
    1. 로그 수익률 변동성(GARCH)
    2. 이동 평균 (ARIMA) 
2. **파생변수(통계 지표) 전처리**
    1. 리스크 (VaR, ES)
        1. VaR 계산 방식 (금융 시장은 t-분포 사용)
            
            ```
            VaR_t = z * σ_t 
            ```
            
    2. Indicator(RSI, BB, MA, Golden Cross)

**계산 주기**

- 1일 단위 (오후 6시 장 종료 후 종가 기준 모델링)

**캐싱 구조** 

- 모델 결과를 서버에 저장
→ 사용자 요청 시 즉시 결과 반환

### 3) Backend → Frontend

python 환경의 모델 분석 결과를 UI로 전송

- **UI 구현**
대시보드 **→** Streamlit
인터랙티브 차트 → plotly

- **결과 제공 방식**
사전 연산된 모델의 dataframe에서 인덱싱

## 5. 기술 스택

### 데이터 수집 및 관리

- `yfinance` API
→ 데이터 수집
- python
→ data 전처리

### 배치 및 계산 관리

- Python Scheduler (예: cron / schedule)
→ 장 종료 후 VaR 자동 계산 (1일 단위)
- 캐싱 (파일 or 메모리 기반)
→ 종목별 계산 결과 저장 및 조회

### Backend

- Python (csv / config 기반)
    - `arch`
    → 모델링
    - numpy / scipy
    → 통계 지표 연산
- LLM API (Llama, gemini 등 무료api)
→ 챗봇 구현

### Frontend

- Streamlit
→ 사용자 인터페이스 및 대시보드 구성
- Plotly
→ 시각화 (인터랙티브 차트)

## 6. 유지/보수

- **적용 종목의 확장과 실시간 서비스 적합**
본 서비스는 모든 주가 종목 뿐 아니라 가상화폐나 유가 등의 차트에도 적용이 가능하며, 일 단위가 아닌 시간이나 초 단위로도 분석 가능함. 그러나 모든 금융 시장의 실시간 주가 차트를 API에서 초 단위로 호출하는 것은 개인(학생)의 하드웨어 환경과 자본으로는 불가능하며, 백엔드 컴퓨팅에도 막대한 시간이 소요된다. 이러한 한계로 본 서비스에서는 우량주 10 종목을 시범적으로 선보인다. 추후 적용 종목 확장과 실시간 서비스 적합을 위해서는 python 파일을 함수 형태로 구성하였기 때문에 API에서 호출된 어느 데이터라도 바로 적용 가능하나, UI의 유저 프로필 설정에서 종목 선택 항목의 간단한 수정이 필요하다.
- **챗봇 모델 변경**
본 서비스의 챗봇은 API로 구현되었다. 차후 더 좋은 성능의 모델이나 유료 모델로 변경할 때에는 호출 API의 변경만으로 간단히 변경 가능하다.
